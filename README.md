# Validar
Laurahacela2porfavor


conectar la base de datos login a un html para ingresar un usuario y una contraseña. si estas son correctas, se muestra "inicio de sesion exitoso". de lo contrario, se muestra "Usuario o contraseña incorrectos. Por favor, inténtalo de nuevo."

Componentes de Bootstrap utilizados
Grid System (container, row, col)
Para centrar el login y hacerlo responsive.
Navbar (navbar)
Barra superior para simular un sistema real.
Card (card, card-header, card-body, card-footer)
Contenedor principal del formulario.
Forms (form-group, form-control, input-group)
Inputs organizados con iconos y mejor presentación.
Buttons (btn, btn-block, btn-lg)
Botón de ingreso grande y adaptable.
Alerts (alert)
Mensaje informativo para el usuario.
Progress (progress, progress-bar)
Barra que simula estado del sistema.
Spinner (spinner-border)
Indicador visual de carga/verificación.
Badges (badge)
Estados del sistema (online, mantenimiento, etc.).
Utilities (clases como text-center, mt-3, p-4)
Para espaciado, alineación y diseño general.
